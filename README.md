### Is the mod compatible with NTM: Extended edition addons/shaders?

Installing EE addons will likely cause crashes and make the modpack unplayable.

The new gun system causes shader incompatibility and heavy visual artifacts when holding a gun.

Shaders also conflict with the NTM skybox.

This can be fixed by editing config/hbm/hbm.cfg and setting:
B:1.00_enableSkybox=false

### How different is it from Extended edition?

**Extended worlds are fully incompatible!** <br>
